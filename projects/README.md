# Projects Directory

Showcase your academic and personal projects here!

## Suggested Structure

```
projects/
├── project-one/
│   ├── README.md
│   ├── images/
│   └── files/
├── project-two/
│   ├── index.html
│   ├── styles.css
│   └── script.js
└── project-three/
    └── README.md
```

## How to Add Projects

1. Create a folder for each project
2. Add a README.md with project description, goals, and results
3. Include relevant files (code, images, documentation)
4. Link from the main website (index.html or content.json)
5. Optionally create a standalone HTML page for each project

## Project Template

```markdown
# Project Title

## Description
Brief description of your project

## Technologies Used
- List technologies here

## Key Features
- Feature 1

- Feature 3

## Results/Outcomes
What did you achieve?

## How to View

```
