# Notes Directory

Add your study materials here! Organize by subject or course.

## Suggested Structure

```
notes/
├── mathematics/
│   ├── algebra.md
│   ├── calculus.md
│   └── statistics.md
├── science/
│   ├── biology.md
│   ├── chemistry.md
│   └── physics.md
├── history/
│   ├── world-history.md
│   └── us-history.md
└── other/
    └── general-notes.md
```

## How to Add Notes

1. Create a markdown (.md) file for each subject or topic
2. Write your notes in markdown format
3. Link them in the main website (index.html or content.json)
4. You can also create HTML files for more complex layouts
